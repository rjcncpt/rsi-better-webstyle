Spectrum Dark Theme



Options

Select the width of the Spectrum forum. 70% is a comfortable width

Description

This extension changes the appearance of Spectrum and some RSI web pages. Important: Dark mode must be activated in Spectrum. Light mode is not supported and will be visually destroyed by this extension.

Stay tuned!
